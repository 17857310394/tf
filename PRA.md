# 塔防游戏产品需求文档 (PRA)

## 1. 产品概览

- **游戏名称**：Escape From The Red Planet
- **游戏类型**：塔防游戏
- **游戏平台**：PC
- **游戏主题**：红色星球科幻主题
- **游戏风格**：3D视角切换塔防，低多边形（low-poly）美术风格
- **核心玩法**：玩家通过建造防御塔、切换视角、参考小地图来抵御敌人的进攻，保护基地安全。

## 2. 核心玩法

### 2.1 游戏机制

- **塔防系统**：玩家可以在指定区域建造不同类型的防御塔，每种塔都有独特的攻击方式和属性。
- **视角切换**：玩家可以在游戏中自由切换不同的视角，包括鸟瞰视角、第一人称视角和第三人称视角，以便更好地观察战场和操作防御塔。
- **小地图系统**：游戏界面中会显示小地图，玩家可以通过小地图了解整个战场的布局和敌人的位置。
- **敌人系统**：游戏中会出现不同类型的敌人，每种敌人都有不同的属性和行为模式。
- **资源系统**：玩家需要通过击败敌人获得资源，用于建造和升级防御塔。

### 2.2 核心循环

游戏的核心循环包括以下步骤：
1. **建造防御塔**：玩家使用资源建造不同类型的防御塔。
2. **抵御敌人**：防御塔自动攻击敌人，玩家可以切换视角观察和操作。
3. **获取资源**：击败敌人获得资源。
4. **升级防御塔**：使用资源升级防御塔，提升其属性。
5. **重复循环**：随着敌人波次的增加，玩家需要不断调整防御策略。

### 2.3 玩家类型分析

- **成就者**：通过完成关卡、解锁新防御塔获得成就感。
- **探索者**：探索不同的防御塔组合和策略。
- **社交者**：与朋友分享游戏进度和策略。
- **杀手**：享受击败敌人和Boss的快感。

### 2.4 游戏流程

1. **关卡选择**：玩家从关卡列表中选择要挑战的关卡。
2. **部署防御**：玩家在指定区域建造防御塔，布置防御阵容。
3. **开始战斗**：玩家点击开始按钮，敌人开始进攻。
4. **实时操作**：玩家在战斗过程中可以切换视角、调整防御塔的位置和攻击目标。
5. **战斗结束**：如果玩家成功抵御所有敌人的进攻，则关卡胜利；如果基地被敌人摧毁，则关卡失败。
6. **奖励结算**：关卡胜利后，玩家会获得相应的奖励，包括资源和解锁新的防御塔。

## 3. 系统架构

### 3.1 核心系统

- **塔防系统**：负责防御塔的建造、升级、攻击等核心功能。
- **视角切换系统**：负责不同视角之间的切换和控制。
- **小地图系统**：负责显示战场的整体布局和敌人的位置。
- **关卡系统**：负责关卡的加载、管理和进度保存。
- **塔防单位系统**：负责防御塔的类型、属性和行为。
- **敌人系统**：负责敌人的类型、属性和行为。
- **资源系统**：负责资源的获取、管理和使用。
- **界面系统**：负责游戏的界面显示和用户交互。

### 3.2 技术架构

- **游戏引擎**：Godot 4.0 或更高版本
- **编程语言**：GDScript
- **开发工具**：Godot 内置编辑器
- **版本控制**：Git
- **渲染系统**：Godot 内置3D渲染系统
- **物理引擎**：Godot 内置物理引擎
- **音效系统**：Godot 内置音效系统

## 4. 功能模块

### 4.1 核心塔防系统

- **防御塔建造**：玩家可以在指定区域点击建造防御塔，选择塔的类型和位置。
- **防御塔升级**：玩家可以花费资源升级防御塔，提升其攻击力、攻击范围和攻击速度。
- **防御塔攻击**：防御塔会自动攻击范围内的敌人，不同类型的塔有不同的攻击方式。
- **防御塔管理**：玩家可以查看和管理已建造的防御塔，包括升级、出售、修复等操作。玩家可以通过点击防御塔打开管理面板，查看防御塔的详细属性和状态。

### 4.2 视角切换系统

- **鸟瞰视角**：玩家可以从空中俯瞰整个战场，了解整体布局和敌人的位置。
- **第一人称视角**：玩家可以切换到某个防御塔的第一人称视角，观察其攻击效果和周围环境。
- **第三人称视角**：玩家可以切换到第三人称视角，自由移动和观察战场。
- **视角控制**：玩家可以通过鼠标和键盘控制视角的移动、旋转和缩放。

### 4.3 小地图系统

- **小地图显示**：游戏界面中会显示小地图，标记玩家的基地、防御塔和敌人的位置。
- **小地图交互**：玩家可以点击小地图上的位置，快速切换视角到该位置。
- **小地图缩放**：玩家可以调整小地图的缩放比例，以便更好地观察战场。

### 4.4 关卡系统

- **关卡设计**：每个关卡都有不同的地图布局、敌人类型和数量。
- **关卡难度**：关卡难度会随着玩家的进度逐渐增加。
- **关卡进度**：玩家可以保存关卡的进度，下次继续挑战。
- **关卡奖励**：关卡胜利后，玩家会获得相应的奖励，包括资源和解锁新的防御塔。

### 4.5 塔防单位系统

- **塔的类型**：游戏中有多种类型的防御塔，包括：
  - **普通塔**：基础攻击塔，攻击力一般，攻击速度快。
  - **导弹塔**：远程攻击塔，攻击力高，攻击速度慢。
  - **激光塔**：持续伤害塔，对单个敌人造成持续伤害。
  - **减速塔**：辅助塔，减慢敌人的移动速度。
  - **范围塔**：范围攻击塔，对范围内的多个敌人造成伤害。
- **塔的属性**：每个塔都有攻击力、攻击范围、攻击速度、生命值等属性。
- **塔的升级**：玩家可以花费资源升级塔，提升其属性。

### 4.6 敌人系统

- **敌人类型**：游戏中有多种类型的敌人，包括：
  - **普通敌人**：基础敌人，生命值低，移动速度慢。
  - **快速敌人**：移动速度快，生命值低。
  - **重型敌人**：生命值高，移动速度慢。
  - **飞行敌人**：可以飞行，避开地面障碍。
  - **Boss敌人**：强大的敌人，生命值高，攻击力强,拥有特殊ai机制，能够根据玩家的位置和状态进行智能决策。

- **敌人弱点系统**：每个敌人都有特定的弱点部位，攻击弱点部位和非弱点部位会造成不同的伤害效果：
  - **弱点部位**：攻击敌人的弱点部位会造成200%的伤害。
  - **非弱点部位**：攻击敌人的非弱点部位会造成100%的伤害（基础伤害）。
  - **弱点配置**：弱点部位的位置和数量因敌人类型而异，不一定是头部，可以根据敌人的设计进行配置。
- **敌人行为**：敌人会沿着预设的路径移动，攻击玩家的基地。
- **敌人波次**：每个关卡都会有多个波次的敌人，波次之间会有短暂的休息时间。

### 4.7 资源系统

- **资源类型**：游戏中有两种主要资源：
  - **金币**：用于建造和升级防御塔。
  - **能源**：用于使用特殊技能和解锁新的防御塔。
- **资源获取**：玩家可以通过击败敌人获得资源。
- **资源管理**：玩家需要合理管理资源，平衡建造和升级防御塔的需求。

### 4.8 界面系统

- **主界面**：显示游戏标题、开始游戏、设置、退出游戏等选项。
- **游戏界面**：显示战场、防御塔、敌人、资源、小地图等信息。
- **小地图**：显示战场的整体布局和敌人的位置。
- **视角切换界面**：显示当前视角和切换视角的按钮。
- **塔防单位选择界面**：显示可建造的防御塔类型和其属性。
- **资源管理界面**：显示当前资源数量和资源获取情况。
- **关卡选择界面**：显示可用的关卡和玩家的进度。
- **设置界面**：显示游戏的设置选项，包括音效、画面、控制等。
- **游戏结束界面**：显示游戏的结果和奖励。

## 5. 界面设计

### 5.1 界面风格

- **整体风格**：科幻风格，符合红色星球的主题，采用低多边形（low-poly）美术风格。
- **色彩方案**：以红色、橙色、黑色为主，突出红色星球的氛围。
- **UI元素**：使用未来感的UI元素，包括全息投影效果、科技感十足的按钮和图标，保持简洁的几何形状。
- **字体**：使用科幻风格的字体，增强游戏的科技感。
- **模型风格**：采用低多边形（low-poly）风格，模型结构简单，细节较少，保持游戏运行流畅。

### 5.2 界面布局

- **主界面**：居中显示游戏标题，下方显示开始游戏、设置、退出游戏等按钮。
- **游戏界面**：
  - 左上角：显示玩家的资源数量。
  - 右上角：显示小地图。
  - 左下角：显示可建造的防御塔类型。
  - 右下角：显示视角切换按钮和游戏控制按钮。
  - 中央：显示战场和敌人。
- **小地图**：位于右上角，显示战场的整体布局和敌人的位置，使用不同颜色标记不同的单位。
- **视角切换界面**：位于右下角，显示当前视角和切换视角的按钮。
- **塔防单位选择界面**：位于左下角，显示可建造的防御塔类型和其属性，点击后可以选择建造。
- **资源管理界面**：位于左上角，显示当前资源数量和资源获取情况。
- **关卡选择界面**：显示可用的关卡和玩家的进度，点击关卡后可以进入游戏。
- **设置界面**：显示游戏的设置选项，包括音效、画面、控制等，玩家可以调整这些设置。
- **游戏结束界面**：显示游戏的结果和奖励，玩家可以选择重新开始或返回主界面。

### 5.3 用户体验

- **操作便捷**：玩家可以通过鼠标和键盘轻松操作游戏，包括建造防御塔、切换视角、控制游戏进程等。
- **反馈及时**：游戏会及时反馈玩家的操作，包括防御塔的建造、升级、攻击效果等。
- **视觉效果**：游戏使用高质量的3D图形，包括流畅的动画、逼真的特效和精美的场景。
- **音效效果**：游戏使用逼真的音效，包括防御塔的攻击声、敌人的叫声、爆炸效果等。
- **游戏节奏**：游戏的节奏适中，玩家可以在战斗中自由调整策略，体验塔防游戏的乐趣。

## 6. 技术需求

### 6.1 硬件需求

- **最低配置**：
  - 操作系统：Windows 10
  - 处理器：Intel Core i5-4460 或 AMD Ryzen 3 1200
  - 内存：8 GB RAM
  - 显卡：NVIDIA GeForce GTX 750 Ti 或 AMD Radeon RX 560

  - 操作系统：Windows 10
  - 处理器：Intel Core i5-8400 或 AMD Ryzen 5 2600
  - 内存：16 GB RAM
  - 显卡：NVIDIA GeForce GTX 1060 或 AMD Radeon RX 580
  - 存储空间：2 GB 可用空间

### 6.2 软件需求

- **游戏引擎**：Godot 4.0 或更高版本
- **操作系统**：Windows 10 或更高版本
- **开发工具**：Godot 内置编辑器
- **版本控制**：Git

## 8. 总结

Escape From The Red Planet 是一款以红色星球为背景的3D视角切换塔防游戏，玩家可以通过建造防御塔、切换视角、参考小地图来抵御敌人的进攻，保护基地安全。游戏具有丰富的塔防单位、敌人类型和关卡设计，玩家可以在游戏中体验到塔防游戏的乐趣和策略性。

游戏的核心特点包括：
- 3D视角切换，提供不同的观察方式
- 小地图参考，帮助玩家了解战场全局
- 丰富的塔防单位和敌人类型
- 红色星球科幻主题，营造独特的游戏氛围
- 流畅的游戏体验和精美的视觉效果

通过合理的开发计划和技术实现，Escape From The Red Planet 有望成为一款受欢迎的塔防游戏，为玩家带来独特的游戏体验。
